## Nomenclature produit

- Ne pas utiliser « crème à café » → utiliser « rehausseur à café »
- « À base de plantes » ne peut pas être utilisé pour les produits International Delight
- Dans les cafés glacés, il n'y a pas de « vrai lait » — ne pas le mentionner

## Allégations sans sucre — formulations approuvées seulement

Utiliser uniquement ces formulations exactes (aucune variation) :
- sans sucre
- 0 sucre
- zéro sucre
- aucun sucre
- ne contient pas de sucre
- 0 g sucre
- zéro g sucre
- 0 gramme sucre
- zéro gramme sucre

## Vocabulaire

- Ne jamais utiliser le mot « treat »
- Symbole de marque déposée : MD en exposant (pas ®)
- Respecter toutes les restrictions dans general_danonecanada.md
